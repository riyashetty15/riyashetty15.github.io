A Brief Description

To comprehend crime patterns and types of crimes, we have looked into several earlier publications that used various visualization methods to depict crime data throughout the US. There are two primary categories for reports on violent and property crimes. Violent crimes include assaults, murders, rapes, and robberies, whereas property crimes include burglaries, robberies, and crimes involving motor vehicles.

The main motivation of this project is to understand what factors contributed to this sudden increase in crime rate and to identify what kinds of crimes are more prevalent in the US.

Technologies / Libraries Used: Seaborn, Plotly, Matplotlib, Python, HTML
